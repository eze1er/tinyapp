## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

### functions 

__<i>we have three functions for this app</i>__

*  urlsForUser
* generateRandomString 
* getUserByEmail

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
